# Painel PM2 Web

Esta aplicação é feita para ser rápida, otimizada e ser self-hosted. Não há foco em ter SEO, na verdade, queremos tirar tudo o que tem a ver com SEO para priorizar a experiência do usuário e a performance.

## Funcionalidades

- Listagem de processos
- Criação de processos via interface
- Edição de processos via interface
- Remoção de processos via interface
- Gerenciamento de arquivos .env de um projeto via interface

Não deve ter autenticação, qualquer um que acessar o painel poderá gerenciar os processos.
